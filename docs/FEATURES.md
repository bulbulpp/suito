# FEATURES.md — suito 機能仕様

## 1. デザイン基調

### 白黒ベース UI

- **ベースカラー**: 純白 `#FFFFFF` / 純黒 `#000000`
- **アクセント**: グレースケール（5 段階）：`#1A1A1A`, `#404040`, `#808080`, `#B3B3B3`, `#E5E5E5`
- **強調色**: 最小限（収入 ＝ 上向き三角 ▲、支出 ＝ 下向き三角 ▼。色は使わず形と濃さで区別）
- **Bitcoin マーク**: ⚡ または ₿ をシンボルとして使用
- **タイポグラフィ**: サンセリフ、数字は等幅フォント（金額表示の縦揃え）
- **角丸最小**：シャープでミニマル
- **ダークモード**: 反転（黒ベース）

### レイアウト指針

- カード積み重ね型は避け、**罫線とスペースで階層**を表現
- ホワイトスペース多め
- 余計なアイコン装飾なし
- meiso の「TeuxDeux 風シンプル」の精神を踏襲

## 2. コア機能（MVP）

### 2.1 取引記録

- **新規取引追加**：金額、カテゴリ、日時、メモ、支払い方法
- **編集・削除**
- **収入 / 支出の切り替え**
- **法定通貨（JPY/USD）と Bitcoin（sat）の両対応**
- **支払い方法**：現金 / カード / 銀行振込 / Bitcoin / その他

### 2.2 カテゴリ管理

#### 支出デフォルト

- 食費 🍴
- 交通費 🚆
- 日用品 🧴
- 雑費 💰
- 光熱費 💡
- 家賃 🏠
- 通信費 📱
- 医療費 🏥
- 娯楽 🎮
- 衣類 👕

#### 収入デフォルト

- 給料 💼
- 臨時収入 🎁
- 投資 📈
- 返金 ↩️
- その他 ✨

#### カスタムカテゴリ

- ユーザーが追加・削除可能
- アイコン（絵文字）と名前を指定
- デフォルトカテゴリも非表示にできる（削除はしない）

### 2.3 カレンダービュー

- **月表示**：各日に **収支合計** を小さく表示
  - 黒太字：支出が多い日
  - 通常字：収入が多い日
  - 何もない日：薄いグレーの「・」
- **日タップ**：その日の取引一覧へ遷移
- **月切り替え**：左右スワイプ or 矢印
- **月合計**：上部に表示（収入・支出・差額）
- **年表示**（将来）：12 ヶ月のミニカレンダー

### 2.4 為替・BTC レート表示

- **画面上部 or 設定画面に常時表示**：
  - 1 BTC = ? 円
  - 1 BTC = ? USD
  - 1 USD = ? 円
- **60 秒キャッシュ**で取引所負荷を軽減
- **手動更新ボタン**
- **オフライン時**は「(最終更新: 〇分前)」と表示

### 2.5 Bitcoin 支払い記録

- **入力モード切り替え**：sat / 円 / USD のいずれかで入力 → 他を自動換算
- **記録時にレート確認画面**：「1 BTC = X 円、Y sat = Z 円ですよ」と表示してから確定
- **取引データに当時レート保存**：後で振り返ったときに当時の換算値が見える
- **取引一覧での区別**：⚡ アイコン + sat 表記をメインに、円換算を副表示

### 2.6 Nostr 同期

- **アカウント**：
  - 新規生成（推奨）
  - 既存 nsec のインポート
- **リレー管理**：
  - デフォルト 3〜5 個のリレーを推奨
  - ユーザーが追加・削除可能
- **同期動作**：
  - 自動同期：起動時 + 取引追加時 + 30 分間隔
  - 手動同期：プルダウン or 設定画面ボタン
- **暗号化**：NIP-44 v2 で自分宛て暗号化（送信者 = 受信者 = 自分）
- **複数端末**：同じ nsec を別端末でインポートすると、リレー経由で同期される

### 2.7 設定画面

- アカウント：公開鍵表示（コピー可能）、秘密鍵エクスポート（明示操作）
- リレー：一覧、追加、削除
- カテゴリ管理：追加、削除、並び替え
- 表示：ダークモード切替、デフォルト通貨
- データ：エクスポート（JSON、暗号化付き）、インポート
- セキュリティ：アプリロック（PIN / 生体認証）

## 3. 将来機能（MVP 後）

### 3.1 レシート OCR

- **カメラでレシート撮影** → 画像取得
- **OCR 処理**：
  - **オンデバイス OCR 優先**（プライバシー上）
  - iOS: `Vision Framework`
  - Android: `ML Kit Text Recognition`（オフライン）
  - これらが扱えない場合のフォールバック検討（要選定）
- **解析**：金額、日付、店名、品目をパース
- **自動入力**：取引フォームに反映、ユーザー確認後保存
- **画像保存**：暗号化済みでローカル保存（オプション）

### 3.2 集計・分析

- **月次レポート**：カテゴリ別円グラフ（白黒の濃淡で表現）
- **年次レポート**
- **平均**：日平均、週平均、月平均
- **比較**：先月比、前年同月比

### 3.3 予算管理

- **カテゴリ別予算設定**
- **超過アラート**
- **残額表示**

### 3.4 Bitcoin 関連の発展

- **BIP-21 / Lightning invoice 読み取り**：QR コードから金額・宛先を取得して記録
- **ウォレット残高表示**（読み取り専用、xpub から）

### 3.5 マルチ通貨対応

- JPY / USD 以外の通貨追加
- 通貨換算履歴

## 5. 操作性

### ジェスチャー

- **取引追加**：FAB（Floating Action Button）or 右下ボタン
- **取引編集**：タップ
- **取引削除**：左スワイプ → 削除ボタン
- **カレンダー**：左右スワイプで月切り替え
- **同期**：プルダウンリフレッシュ

### キーボード

- **金額入力**：数値キーボード（10 キー）+ カスタム単位切替ボタン
- **メモ入力**：通常キーボード
- **コピー＆ペースト**：取引のレートはコピー可能

### アクセシビリティ

- **VoiceOver / TalkBack 対応**：すべての UI 要素にラベル
- **フォントサイズ追従**：OS 設定に従う
- **コントラスト比**：WCAG AA 以上

## 6. 多言語

- **デフォルト**: 日本語
- **対応**: 日本語 / 英語（最低限）
- **将来**: 中国語、スペイン語、ポルトガル語等

## 7. パフォーマンス目標

- **起動時間**：コールド起動 3 秒以内
- **画面遷移**：すべて 200ms 以内
- **同期処理**：UI ブロックしない（バックグラウンド）
- **取引一覧**：1000 件まで快適にスクロール

## 8. ストレージ容量見積もり

- 1 取引 ≒ 500 バイト（freezed JSON + 暗号化オーバーヘッド）
- 月 100 取引 × 5 年 = 6000 取引 ≒ 3 MB
- レシート画像（将来）：取引 1 件あたり最大 500 KB
- 想定総容量：100 MB 以内に収める

## 9. オフライン動作

- **完全オフライン**で取引追加・閲覧・編集・削除可能
- **オンライン復帰時**に自動同期
- **競合解決**：タイムスタンプ最新優先（CRDT は将来検討）
